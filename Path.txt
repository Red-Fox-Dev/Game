from queue import PriorityQueue เป็นการ import คลาส PriorityQueue จาก module queue ใน Python ซึ่งมีความสำคัญดังนี้:

PriorityQueue คืออะไร?


เป็นโครงสร้างข้อมูลแบบคิวพิเศษที่เรียงลำดับข้อมูลตามความสำคัญ
แต่ละรายการจะมีค่าลำดับความสำคัญ (priority)
รายการที่มีลำดับความสำคัญสูงสุดจะถูกนำออกมาก่อน


วิธีใช้งานพื้นฐาน:

pythonCopyfrom queue import PriorityQueue

# สร้าง PriorityQueue
pq = PriorityQueue()

# เพิ่มข้อมูล (priority, item)
pq.put((2, 'มะนาว'))   # priority 2
pq.put((1, 'แอปเปิ้ล')) # priority 1
pq.put((3, 'กล้วย'))   # priority 3

# นำข้อมูลออก
print(pq.get())  # จะได้ (1, 'แอปเปิ้ล')



A* Algorithm (อ่านว่า "A-star") เป็นอัลกอริทึมที่ใช้ในการค้นหาเส้นทางที่สั้นที่สุดและมีประสิทธิภาพ มีลักษณะสำคัญดังนี้:

หลักการทำงาน:


รวมระยะทางที่ผ่านมาแล้ว (actual cost) กับการประมาณระยะทางที่เหลือ (heuristic estimate)
เลือกเส้นทางที่มีต้นทุนรวมต่ำสุด


สูตรคำนวณ:
f(n) = g(n) + h(n)


f(n) = ต้นทุนรวม
g(n) = ระยะทางที่ผ่านมาแล้ว
h(n) = การประมาณระยะทางที่เหลือ


ขั้นตอนการทำงาน:


เริ่มจากจุดเริ่มต้น
สำรวจเส้นทางรอบข้าง
เลือกเส้นทางที่มีต้นทุนรวมต่ำสุด
ทำซ้ำจนกระทั่งถึงจุดหมาย
